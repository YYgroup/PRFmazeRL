# PRFmazeRL
Code for SAC control of a 2D jellyfish that can navigate in environment with obstacles.

Training dataset available at https://huggingface.co/datasets/YYgroup/PRFmazeRL/tree/main

Dataset processing method and CFD tool are the same as in https://www.cambridge.org/core/journals/journal-of-fluid-mechanics/article/abs/deep-reinforcement-learning-for-tracking-a-moving-target-in-jellyfishlike-swimming/195BEA081AB8A0435B4C05908BF474CE
